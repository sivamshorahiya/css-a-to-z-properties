Introduction

Cascading Style Sheets (CSS) is a cornerstone technology in web development, allowing developers to control the presentation and layout of web pages. This document serves as a comprehensive guide to important CSS properties, organized in a dictionary-style format for easy reference.

a Properties

  • animation:

    The animation property in CSS can be used to animate elements in a more interesting and complex manner. It allows for the animation of multiple properties at the same time, and can be used to create complex animations with keyframes.

    Example:

                      .element {
                    width: 100px;
                    height: 100px;
                    background-color: red;
                    position: relative;
                    animation-name: example;
                    animation-duration: 4s;
                  }
                  @keyframes example {
                    0%   {left: 0px; top: 0px;}
                    25%  {left: 200px; top: 0px;}
                    50%  {left: 200px; top: 200px;}
                    75%  {left: 0px; top: 200px;}
                    100% {left: 0px; top: 0px;}
                  }

  • align-content:

    • Used in flexbox and grid layouts, this property aligns the flex lines or grid tracks within the content area of a flex or grid container when there's extra space

    Example:

                    .container {
                  display: flex;
                  flex-wrap: wrap;
                  align-content: space-between;
                }

  • align-items:

    • Used in flexbox and grid layouts, this property aligns the items within the flex container or grid container along the cross axis, which is the vertical axis for a row-based flex container, and the horizontal axis for a column-based flex container.

    Example:

                    .container {
                  display: flex;
                  align-items: center;
                }

  • align-self:

    • Used in flexbox and grid layouts, this property aligns a single item in the flex container or grid container along the cross axis, overriding the align-items value.

    Example:

                    .item {
                  align-self: flex-end;
                }

  • all:

    • The all property in CSS resets all properties to their initial values, or in the case of shorthands, to their individual initial values.

    Example:

                    .element {
                  all: initial;
                }
b Properties

  • backdrop-filter:

    • The backdrop-filter property in CSS applies filter effects to the area behind an element. This property is often used to create frosted glass effects, particularly on top of background images.

    Example:

                    .element {
                  backdrop-filter: blur(10px);
                }

  • background:

    • The background property in CSS is used to set the background of an element. It can be used to set the background color, image, and position, among other properties.

    Example:

                    .element {
                  background: #f0f0f0 url('image.jpg') no-repeat center center;
                }

  • background-attachment:

    • The background-attachment property in CSS specifies whether the background image is fixed with regard to the viewport, or scrolls along with the element's contents.

    Example:

                    .element {
                  background-attachment: fixed;
                }

  • background-clip:

    • The background-clip property in CSS specifies whether the background image of an element should be clipped to the element's border box, padding box, or content box.

    Example:

                    .element {
                  background-clip: padding-box;
                }

  • background-color:

    • The background-color property in CSS is used to set the background color of an element.

    Example:

                    .element {

  • background-origin:

    • The background-origin property in CSS specifies the positioning area of the background images in an element.

    Example:

                    .element {

                }

  • background-position:

    • The background-position property in CSS is used to set the initial position of a background image. It can be used to position the image horizontally and vertically within the element's background area.

    Example:

                    .element {
                  background-position: center center;

  • background-repeat:

    • The background-repeat property in CSS is used to specify how a background image should be repeated within an element's background area.

    Example:

  • background-size:

    • The background-size property in CSS is used to specify the size of a background image. It can be used to stretch or shrink the image to fit the element's background area.

    Example:

                    .element {
                  background-size: cover;

  • block-size:

    • The block-size property in CSS is used to set the size of the block-level axis of an element. It is often used to set the height of an element, depending on the writing mode of the document.

    Example:

                    .element {

    Example:

    • The border-collapse property in CSS is used to specify whether the table's border and the cells' borders are collapsed into a single border or separated.

    Example:

                    table {
                  border-collapse: collapse;
                }

    • The border-spacing property in CSS is used to set the distance between the borders of adjacent cells in a table.

    Example:

    • The bottom property in CSS is used to set the bottom position of a positioned element.

    Example:

  • box-shadow:

    • The box-shadow property in CSS is used to add shadow effects around an element's frame. It can be used to cast a drop shadow from the frame of the element, and can be used to create a shadow effect in or out of the frame.

    Example:

                    .element {

    • The box-sizing property in CSS is used to alter the default CSS box model used to calculate width and height of the elements.

c Properties

    Example:

                    .element {
                  color: #ff0000;
                }

  • clear:

    • The clear property in CSS is used to specify whether an element should be moved below (cleared) floating elements that precede it.

    Example:

                    .element {
                  clear: both;
                }

  • cursor:

    • The cursor property in CSS is used to specify the type of cursor to be displayed when pointing over an element.

    Example:

                    .element {
                  cursor: pointer;
                }

  • clip:

    • The clip property in CSS is used to specify a visible region of an element. It is often used to hide a part of an element that is not needed.

    Example:

                    .element {
                  position: absolute;
                  clip: rect(0px,60px,200px,0px);
                }

  • content:

    • The content property in CSS is used to insert generated content before or after an element's content.

    Example:

                    .element::before {
                  content: "Hello";
                }

  • counter-*:

    • The counter-* properties in CSS are used to create and control counters, which are used to increment or decrement values associated with elements.

    Example:

                    .element::before {
                  counter-increment: my-counter;
                  content: counter(my-counter);
                }

  • column-*:

    • The column-* properties in CSS are used to control the layout of content in multi-column elements.

    Example:

                    .element {
                  column-count: 3;
                  column-gap: 20px;
                }

  • columns:

    • The columns property in CSS is used to specify the number of columns an element should be divided into and the width of the columns.

    Example:

                    .element {
                  columns: 3 200px;
                }

  • caret-color:

    • The caret-color property in CSS is used to set the color of the cursor (caret) in inputs, textareas, or any element that is editable.

    Example:

                    input {
                  caret-color: red;
                }

  • content-visibility:

    • The content-visibility property in CSS is used to control the rendering behavior of an element's content, allowing it to be skipped or rendered lazily.

    Example:

                    .element {
                  content-visibility: auto;
                }

  • contain:

    • The contain property in CSS is used to indicate that the element's subtree is independent of the rest of the page, and can be treated as a single unit.

    Example:

                    .element {
                  contain: layout;
                }
d Properties

    • The display property in CSS is used to specify the type of rendering box used for an element. It can be used to change the default behavior of an element, such as making it a block-level or inline-level element.

    Example:

                    .element {
                  display: block;
                }

  • direction:

    • The direction property in CSS is used to specify the direction of the text content of an element. It can be used to change the direction of the text from left-to-right to right-to-left, or vice versa.

    Example:

                    .element {
                  direction: rtl;
                }

  • empty-cells:

    • The empty-cells property in CSS is used to specify whether or not to display borders and background on empty cells in a table.

                    table {

f Properties

  • flex:

    • The flex property in CSS is used to specify the ability of a flex item to alter its dimensions to fill the available space. It can be used to set the initial size of a flex item, and to allow it to grow or shrink to fit the available space.

    Example:

                            .item {
                          flex: 1;
                        }

  • flex-basis:

    • The flex-basis property in CSS is used to specify the initial size of a flex item, before any available space is distributed according to the flex factors.

    Example:

                            .item {
                          flex-basis: 100px;
                        }

  • flex-direction:

    • The flex-direction property in CSS is used to specify the direction of the main axis of a flex container. It can be used to set the direction of the flex container as row or column.

    Example:

                            .container {
                          flex-direction: row;
                        }

  • flex-flow:

    • The flex-flow property in CSS is a shorthand property for setting the flex-direction and flex-wrap properties of a flex container.

    Example:

                            .container {
                          flex-flow: row wrap;
                        }

  • flex-grow:

    • The flex-grow property in CSS is used to specify the ability of a flex item to grow and fill the available space in a flex container. It determines how much the flex item will grow relative to other flex items.

    Example:

                            .item {
                          flex-grow: 1;
                        }

  • flex-shrink:

    • The flex-shrink property in CSS is used to specify the ability of a flex item to shrink and occupy less space in a flex container. It determines how much the flex item will shrink relative to other flex items.

    Example:

                            .item {
                          flex-shrink: 0;
                        }

  • float:

    • The float property in CSS is used to specify whether an element should float to the left, right, or not at all. It is commonly used to create layouts where elements are positioned side by side.

    Example:

                            .element {
                          float: left;
                        }

  • font-family:

    • The font-family property in CSS is used to specify the font family or list of font families for text content. It determines the typeface or font face that will be used to render the text.

    Example:

                            .element {
                          font-family: Arial, sans-serif;
                        }

  • font-size:

    • The font-size property in CSS is used to specify the size of the font for text content. It determines the height of the characters in the text.

    Example:

                            .element {
                          font-size: 16px;
                        }

  • font-style:

    • The font-style property in CSS is used to specify the style of the font for text content. It can be used to make the text italic, oblique, or normal.

    Example:

                            .element {
                          font-style: italic;
                        }

  • font-weight:

    • The font-weight property in CSS is used to specify the weight or thickness of the font for text content. It can be used to make the text bold, normal, or lighter.

    Example:

                            .element {
                          font-weight: bold;

    Example:

                  display: grid;

    • The grid-area property in CSS is used to specify a grid item's size and location within a grid layout, and to assign a name to the item for use with grid-template-areas.

    Example:

                  grid-area: header;
                }

    • The grid-auto-columns property in CSS is used to specify the size of the implicit grid columns in a grid layout.

    Example:
